# portal-iqy-hub

## 项目简介

portal-iqy-hub 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面的存放与展示集合。

## 目录说明

```
portal-iqy-hub/
├── index.html          # 主入口页面
├── pages/              # 独立页面存放目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 公共资源
│   ├── css/
│   ├── js/
│   └── images/
└── README.md
```

- `index.html`：仓库的主入口页面，提供页面导航或索引功能。
- `pages/`：存放所有独立 HTML 页面，按需命名或编号。
- `assets/`：存放页面共用的样式、脚本和图片资源。

## 页面归档说明

仓库中的每个 HTML 页面均为独立文件，可直接通过浏览器打开或部署至静态托管服务。页面内容可能涉及不同主题或用途，彼此之间无强关联。所有页面均不包含指向特定外部网站的链接或依赖。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面。
- 不保证页面内容的长期可用性或兼容性。
